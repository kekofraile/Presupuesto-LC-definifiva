# Cómo visualizar la app offline de presupuestos

1. Instala dependencias:
   ```bash
   npm install
   ```
2. Inicia el servidor de desarrollo Vite:
   ```bash
   npm run dev
   ```
3. Abre el navegador en la URL que muestre la consola (por defecto `http://localhost:5173`).
4. Instala la app como PWA desde el menú del navegador para probar el modo offline.
5. Carga tu `tarifa.xlsx` o `tarifa.csv` desde el botón **Actualizar tarifa** y accede con el PIN configurado.
